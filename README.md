This repository contain the source codes for all numerical studies conducted in the paper  
**_Some design considerations for cluster randomized trials with binary responses_** authored by **_Arpan Singh, Meghendar Pal, Satya Prakash Singh_.** For a through reading, refer to the [PAPER](https://www.tandfonline.com/doi/abs/10.1080/03610926.2021.1990954).
